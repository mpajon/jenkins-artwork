## Atribución y Reconocimientos

Este repositorio contiene, entre otros elementos, una imagen representativa del **proyecto Jenkins**, utilizada con el máximo respeto a los derechos de autor y a la comunidad que lo respalda.

### Sobre la imagen utilizada

La imagen, conocida comúnmente como el **logo oficial de Jenkins**, no es una simple ilustración: es un símbolo que representa a una de las herramientas de integración continua más importantes y extendidas en el ámbito del desarrollo de software. Su presencia en este repositorio responde al deseo de rendir homenaje al proyecto Jenkins y facilitar la identificación visual cuando el contexto así lo requiere.
